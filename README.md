A simple vowel counting application built with Vanilla JavaScript. This project focuses on string manipulation and array operations.

🌟 Key Features
Real-time Counting: Updates the count instantly as you type using the keyup event.

Case Insensitive: Supports both uppercase and lowercase characters via .toLowerCase().

Clear Visuals: Large, easy-to-read counter with dynamic UI updates.

🧠 Key Learnings
String Normalization: Using .trim() and .toLowerCase() to standardize user input.

Array Methods: Utilizing .includes() to efficiently check characters against a vowel list.

Iteration: Implementing for...of loops to iterate through string characters.

DOM Manipulation: Updating the UI dynamically based on logic processing.
